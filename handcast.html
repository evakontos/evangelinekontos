"Grasp"

"Grasp", plaster hand cast sculpture on wooden base


This sculpture is a plaster hand cast of both my right and left hands. I chose different hand gestures for each hand which were meant to evoke feeling from the viewer. I created a wooden base and mounted the hands on top. This project was done in collaboration with a partner, where each helped create and set molds for casting. Plaster was then poured into the molds and carefully carved out.
